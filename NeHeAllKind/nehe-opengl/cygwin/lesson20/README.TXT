README for lesson20
===================

The Makefile assumes you have GLUT3.7 installed as follows:

Includes:	/usr/local/include/

Library:	/usr/local/lib/

If your setup is different from this, edit the Makefile accordingly.

F1 key toggles fullscreen/window mode

m/M toggles masking

o/O toggles the top mask

ESC to exit

To make simply type "make"

Enjoy.
