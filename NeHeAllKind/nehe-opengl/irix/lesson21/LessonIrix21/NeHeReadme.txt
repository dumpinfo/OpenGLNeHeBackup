OpenGL Tutorial #21.

Project Name: NeHe's Line, Anti-Aliasing, Timing, Orthographic View and Sound Effect Tutorial

Project Description: Miniature Game Showing Off Multiple Features Of OpenGL

Authors Name: Jeff Molofee (aka NeHe)
IRIX port:    Dimi Christopoulos (20/03/2002)
              (www.geocities.com/dimi_christop)
Changes:
1) Frames per second calculation and display.
1) Implemented better delaying algorithm for faster machines,
by restricting (locking) to specified Frames Per Second (see source code at Update() callback)).
2) F1 to toggle to fullscreen and back.
3) Unfortunately there is no easy way to play sound so the sounds were removed.
Compilation:
To compile change in the makefile the paths to your Glut 3.7 librarys and include files.
Glut 3.7 has to be installed.
